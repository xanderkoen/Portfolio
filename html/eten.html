Eten op rolletjes is een webapplicatie voor mensen met een motorische beperking.

eten op rolletjes is in samenwerking gemaakt met : Mikael Peijer, Ryan de Man en Yaran de Jong

Het was de bedoeling om een webapplicatie te maken die de ervaring uit eten gaan naar een restaurant moest verbeteren.


In de groep hebben we veel ons best gedaan. We hebben doelgroeps onderzoek gedaan, user testing, en om feedback gevraagd
We hebben de webapp interactief gemaakt door een leuke homepage te maken met plaatjes waar je op kan klikken. Iedereen heeft een eigen pagina gemaakt mt daarop zijn informatie om de uit eten ervaring te verbeteren.

Voor de webapp hebben we een API moeten maken die data via JSON exporteerd. Vervolgens moesten we via een fetch request die data importen en gebruiken op onze web applicatie

De huidige pagina maakt helaas geen gebruik van die API aangezien die niet gehost is maar de code is wel terug te vinden in oudere commits.
