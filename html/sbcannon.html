Game Development

Onze game hebben wij ontwikkeld met de Unity Game engine. Unity gebruikt C# en was volledig nieuw voor mij, Gellukig is het vrij gelijk aan de talen die ik al goed ken dus was het makkelijk voor mij om gewend te raken aan de taal. Ik werkte samen met Kawish Gangadin, Youri de Gier, Joeri Göbel en Ali el Metmari. In het team was ik een Developer en heb een meerdere dingen gewerkt in de game.

Game Trailer

Voor de eerste helft van mijn Game Development Minor hebben we een mobile game gemaakt. Spiritbound: Cannonball is een creatieve platformer waar je jezelf naar voren moet lanceren om het terrein te navigeren. Pas op voor gevaarlijke tegenstanders of dodelijke valkuilen. Kan jij ontsnappen uit limbo?
